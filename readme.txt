LEIA ME

Projeto inspirado na OmniStack 8º, veja as vídeos aulas aqui: https://rocketseat.com.br/week-8/aulas#day-05

Será necessário que instale:
Node.js, na versão LTS, o link para download é: https://nodejs.org/en/
Yarn, sem o node, baixe pelo o link: https://yarnpkg.com/pt-BR/docs/install#mac-stable

Neste momento confira se tudo foi instalado, primeiramente reinicie o terminal e então rode:
npm --version
node -v
yarn -v

Se todos retornarem as versões então está pronto para a segunda parte!

Ao instalar o VSCode pelo o link: https://code.visualstudio.com/
Será necessário que instale algumas extensões, que são:
    material icon theme,
    Rocketseat ReactJS,
    Rocketseat React Native.

Baixe a fonte FiraCode (https://github.com/tonsky/FiraCode) e instale.
Faça a configuração da fonte para que os simbolos se unam!
    ctrl + shift + p -> vá para settings.json
    {
        "editor.fontFamily": "Fira Code",
        "editor.fontLigatures": true,
    }
    Deixe as suas preferencias assim!

Agora é só começar a programação!
